# git

## 创建使用仓库

工作区—git add—暂存区(index)—git comman—本地仓库（object)

未跟踪—git add—未修改—修改文件—已修改—git add—已缓存

git init创建仓库

git status查看仓库状态

git add 添加到暂存区

git commit 提交//*git commit -m <文件名>直接提交文件，git commit 文件进入vi编辑模式，可以输入提交备注。*

###整个系统是Linux形式，可以使用git add * 提交多个文件//git add . 

git log 查看提交记录
