#### 排序算法

快速排序：

1. 快速选取中间元素作为基准点；

2. i从左向右找大于等于基准点的元素；

3. j从右向左找小于等于基准点的元素；

4. 两个元素进行互换；

5. 从左向右找大于等于基准点的元素；

6. 从右向左找小于等于基准点的元素；

7. 两个元素进行互换；

8. ...当i>j时第一趟结束；

9. 依次往下排序完成

   void quick_sort(int a[],int left,int right){
       int c=(int)(right+1)/2;
   
       int swapl=0;
       int swapr=0;
       do
       {
           int *l=&a[left];
           int *r=&a[right];
           if (*l>=a[c])
           {
               //l=&a[left];
               swapl=1;
           }
           else{
               left++;
               swapl=0;
           }
   
           if (a[right]<=a[c])
           {
               //r=&a[right];
               swapr=1;
           }
           else{
               right--;
               swapr=0;
           }
           if(swapl==1&&swapr==1){
               int temp;
               temp=*l;
               *l=*r;
               *r=temp;
           }  
       } 
       while (left<right);
   }
